
# ionic-icons-sass
Ionic Icons Convertable to Sass/Scss

## Installation
Install via NPM:

```bash
npm install ionic-icons-sass

```

```scss
@import '~ionic-icons-sass/dist/ionic-icons';
```

```php
<i class="ionic icon-users"></i>
```
